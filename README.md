## Technology Stack

- Language: JavaScript
- Frontend Library: React.JS
- State Management: Redux.JS
- UI Component: Tailwind CSS
- Web Server: Node.JS
- Backend Framework: Express.JS
- Database: MongoDB, Mongoose (ODM)
- CI/CD: Github, Heroku, Firebase.
